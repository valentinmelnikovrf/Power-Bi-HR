# Power-Bi-HR
HR Analytics Dashboard (Power BI)

![HR Свод - Главная страница](https://github.com/valentinmelnikovrf/Power-Bi-HR/raw/main/%D0%A1%D0%B2%D0%BE%D0%B4%201.PNG)

Комплексный анализ персонала с детализацией по демографии, текучести и обучению

**📌 О проекте**

Этот проект представляет собой профессиональный HR-дашборд, разработанный в Power BI Desktop для глубокого анализа кадровых процессов. Панель позволяет HR-специалистам и руководству компании отслеживать ключевые метрики персонала в реальном времени, выявлять проблемные зоны в удержании сотрудников и оценивать эффективность обучения.

**Цель создания:** Предоставить единый источник правды для отдела персонала, автоматизировать сбор отчетности и перейти от разрозненных данных к интерактивной визуальной аналитике.

**Ключевые особенности (Why we are cool) 🚀**
Многостраничная навигация: Реализована удобная панель навигации (HR Свод, Демография, Обучение) для быстрого переключения между контекстами.

Интерактивные срезы: Все страницы объединены глобальными фильтрами (Дата, Департамент, Должность, Стаж), что обеспечивает согласованность данных при анализе.

Комплексные HR-метрики: В дашборде рассчитываются как стандартные, так и продвинутые KPI: уровень текучести (51,1%), вовлеченность (2,94), удовлетворенность (3,02), стоимость обучения на сотрудника.

Детальная демография: Анализ персонала в разрезе пола, возраста, расы и типа занятости с процентным распределением.

Геопространственный анализ: Интегрирована карта (TomTom) для визуализации распределения сотрудников и гендерного баланса по городам присутствия (Москва, Санкт-Петербург, Новосибирск и др.).

Анализ причин увольнений: Разобраны причины уходов (Voluntary, Involuntary, Retirement) в разрезе демографических групп для построения стратегии удержания.

Оценка эффективности обучения: Рассчитаны затраты на тренинги ($1,76M), средняя стоимость тренинга ($561) и процент прохождения (50,41%) с динамикой по статусам (Completed, Failed, Passed).


**Страницы отчета 📄**
**1. HR Свод (Executive Summary)**
Главная страница для быстрой оценки ситуации.

Ключевые показатели: Карточки с общим количеством активных сотрудников (1466), общим % текучести и средними баллами вовлеченности/удовлетворенности.

Текучесть по департаментам: Горизонтальная гистограмма, показывающая уровень текучести в разрезе отделов. Самая высокая текучесть — в IT/IS (79,2%) и Executive Office (60,0%).

Динамика увольнений: Линейная диаграмма (или гистограмма) с трендом увольнений по годам (пик в 2022 году — 253 увольнения?).

Кнопка управления: Реализована кнопка "Очистить все срезы" для сброса фильтров одним кликом.

**Блок «Выбор показателя».**

Это не просто набор кнопок, а современный свитч (срез на основе параметра поля), который управляет всей визуализацией.

Пользователь может одним кликом переключаться между метриками:

Активные сотрудники

Вовлеченность

Текучесть %

Удовлетворенность

Work-Life баланс

Техническая реализация:

Параметры поля (Field Parameters): В модели данных создан параметр, который объединяет различные меры (например, [Активные сотрудники], [Текучесть %], [Вовлеченность]) в единое поле для визуализации.

Единый график: На холсте размещен всего один график («Динамика увольнений» / «Динамика показателей»), который получает значения из этого параметра. Это делает отчет минималистичным, но невероятно информативным.




**2. Демография (Demographics)**
Глубокий анализ состава персонала.

Поло-возрастная структура: Анализ сотрудников по возрастным группам (20-29, 30-39, 40-49, 50+) с разбивкой по полу.

Расовый состав: Таблица с распределением сотрудников по расам (White, Black, Asian, Hispanic, Other) и их долей в компании.

Типы занятости: Анализ соотношения Full-Time, Part-Time и Contract сотрудников.

Гендерная карта: Визуализация на карте России с разделением по полу в ключевых городах.

Матрица "Причины увольнений": Детальный анализ причин ухода в разрезе рас и пола.

**3. Обучение (Learning & Development)**
Анализ эффективности тренингов и затрат на обучение.

Общие инвестиции: Карточки с суммой затрат, количеством тренингов и процентом прохождения.

Эффективность по департаментам: Таблица, показывающая затраты на тренинги в разрезе отделов, среднюю стоимость тренинга и % прошедших обучение.

Динамика тренингов: Временной ряд, отслеживающий количество сотрудников со статусами "Passed", "Failed", "Incomplete" по месяцам.


Используемые технологии и навыки 💡
Инструмент: **Microsoft Power BI Desktop.**

Источники данных: **ODBC MySQL**

Трансформация данных: **Power Query (M-язык) для очистки и подготовки данных о персонале.**

Моделирование данных: **Создание связей между таблицами (Сотрудники, Увольнения, Обучение).**

**DAX (ключевые меры):**

Расчет динамических показателей (Total Employees, Headcount).

Расчет соотношений (Gender Ratio, Turnover Rate %).


**Визуализация:**

Пользовательская навигация с помощью кнопок и закладок.

Использование карт (TomTom).

Сложные таблицы с цветовой индикацией.

**Как пользоваться дашбордом**

Скачайте файл .pbix.

Откройте его в Power BI Desktop (рекомендуется последняя версия).

Используйте панель слева для навигации по страницам.

Применяйте срезы в верхней части страницы ("Дата", "Департамент", "Должность", "Стаж") для фильтрации данных на всех страницах отчета.

Нажимайте на кнопку "Очистить все срезы", чтобы сбросить фильтры.
